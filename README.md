# Elemental Garden 🌱⚡

An interactive neon ecosystem where elemental seeds grow into particle-emitting flora, shaped by gravity wells you create.

## Play

[Play on rexuvia.com](https://rexuvia.com/games/elemental-garden.html)

## Features

- **4 Elements:** Fire 🔥, Ice ❄️, Lightning ⚡, Void 🌀
- **Dynamic Growth:** Plants grow with organic bezier curves that respond to gravity in real-time
- **Gravity Wells:** Drag to create attractive or repulsive gravity fields that bend plants and particles
- **Particle Physics:** Elements emit particles that interact:
  - Fire + Ice = steam/mist
  - Lightning + Void = plasma arcs
  - Fire + Lightning = explosions
  - Ice + Void = crystallization
  - Same element = amplification
- **Cross-Pollination:** Plants create hybrids when their particles collide
- **Generative Audio:** Web Audio synthesis with unique frequencies for each element

## Controls

- Click/tap to plant seeds (select element first)
- Switch to Gravity mode and drag to create gravity wells
- Right-click/long-press gravity wells to toggle attractive ↔ repulsive
- Pause, Clear, and Sound controls available

## Tech

- Vanilla JavaScript with Canvas API
- Self-contained single HTML file
- Mobile-first responsive design
- 60fps animations with particle systems
